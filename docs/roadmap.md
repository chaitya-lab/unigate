# Roadmap

## Phase 1

Architecture reset and new base contracts.

- universal `Message`
- channel contract
- capabilities
- lifecycle types
- kernel events
- exchange skeleton

## Phase 2

Storage interfaces and first backends.

- inbox
- outbox
- session
- secure store
- dedup
- memory
- SQLite

## Phase 3

Exchange kernel pipeline.

- receive
- store
- forward
- retry
- backpressure

## Phase 4

Instance manager and extensions.

## Phase 5

Router / ASGI, CLI, and simple adapters.

## Phase 6

Real external adapters and operational polish.
